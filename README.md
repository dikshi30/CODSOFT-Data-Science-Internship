# CODSOFT Data Science Internship Projects

## Overview

This repository contains the projects completed during my Data Science Internship at CODSOFT. These projects demonstrate the application of data analysis, machine learning, predictive modeling, and data visualization techniques using Python and industry-standard data science libraries.

## Projects

### 1. Titanic Survival Prediction

* Built a machine learning model to predict passenger survival on the Titanic dataset.
* Performed data cleaning, feature engineering, and exploratory data analysis.
* Applied classification algorithms and evaluated model performance.

### 2. Movie Rating Prediction

* Developed a predictive model to estimate movie ratings based on various features.
* Conducted data preprocessing and feature selection.
* Evaluated model accuracy using regression techniques.

### 3. Iris Flower Classification

* Implemented a classification model to identify iris flower species.
* Utilized exploratory data analysis and visualization techniques.
* Trained and evaluated machine learning models for accurate classification.

### 4. Sales Prediction

* Analyzed historical sales data to forecast future sales performance.
* Applied data preprocessing and predictive analytics techniques.
* Generated insights to support business decision-making.

### 5. Credit Card Fraud Detection

* Built a machine learning model to identify fraudulent transactions.
* Addressed class imbalance and performed feature analysis.
* Evaluated model effectiveness using classification metrics.
